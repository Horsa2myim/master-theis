\begin{cabstract}
当前人工智能系统大多依赖数据中的统计相关性，缺乏可解释性与鲁棒性。因果发现旨在从观测数据中揭示变量间内在作用机制，是推动人工智能从“关联驱动”向“因果驱动”演进的关键。然而，现有基于连续优化和知识融合的因果发现方法多聚焦于有向无环图（DAG）结构学习，忽视了对结构因果模型中因果函数的刻画，且难以融合先验知识处理复杂非参数数据。鉴于此，本文引入 Kolmogorov-Arnold 网络（KAN），利用其符号化潜力与函数分解特性，研究基于 KAN 的因果结构与函数联合发现方法。主要研究内容如下：

首先，提出基于 KAN 的连续优化因果发现算法（DAG-KAN）。该算法构建轻量级单层 KAN 架构，实现因果图结构与函数映射的端到端联合学习。通过 KAN 边稀疏性约束学习因果邻接矩阵，利用可学习 B 样条基函数显式拟合变量间非线性因果机制。实验表明，该方法在加性模型数据上的结构学习精度优于主流方法，高维复杂场景下结构汉明距离平均降低 22.6\%。

其次，提出基于 KAN 的知识融合因果发现算法（Causal-KAN）。针对真实场景普遍存在的非参数或半参数数据，设计知识嵌入层 - 函数拟合层解耦架构。通过在输入层引入因果掩码机制，将部分已知因果父节点关系硬编码为网络约束，在保障因果忠实性的同时释放深层网络函数逼近能力。Sachs 真实数据集实验表明，嵌入因果知识可有效防止过拟合，且学习到的因果函数具有可解释性。

最后，总结全文并展望。本文提出的 DAG-KAN 与 Causal-KAN 方法，将因果发现研究范式从因果存在性判断推进至因果作用机制解析，构建了兼具数学透明性与实用鲁棒性的因果函数学习方法，为精准医疗、政策干预等需理解如何干预的领域提供了可操作的因果知识。
\end{cabstract}

\begin{eabstract}
Contemporary artificial intelligence systems predominantly rely on statistical correlations in data, often lacking interpretability and robustness. Causal discovery aims to reveal intrinsic mechanisms between variables from observational data, serving as a pivotal driver for transitioning AI from "correlation-driven" to "causation-driven". However, existing continuous optimization-based causal discovery methods primarily focus on Directed Acyclic Graph (DAG) structure learning, often overlooking the characterization of causal functions within Structural Causal Models (SCM). Furthermore, these methods struggle to effectively integrate prior knowledge when handling complex non-parametric data. To address these challenges, this thesis introduces Kolmogorov-Arnold Networks (KAN), leveraging their innate symbolic potential and function decomposition properties to investigate KAN-based joint discovery methods for causal structure and function. The main research contributions are as follows:

Firstly, a KAN-based continuous optimization causal discovery algorithm (DAG-KAN) is proposed. This algorithm constructs a lightweight single-layer KAN architecture to achieve end-to-end joint learning of causal graph structure and causal function mapping. It learns the causal adjacency matrix through sparsity constraints on KAN edges and explicitly fits nonlinear causal mechanisms between variables using learnable B-spline basis functions. Experiments demonstrate that this method outperforms existing mainstream methods in structure learning accuracy on additive model data, reducing the Structural Hamming Distance by an average of 22.6\% in high-dimensional complex scenarios.

Secondly, a KAN-based knowledge-fused causal discovery algorithm (Causal-KAN) is proposed. Addressing the non-parametric or semi-parametric data prevalent in real-world scenarios, this algorithm designs a decoupled "knowledge embedding layer - function fitting layer" architecture. By introducing a causal masking mechanism at the input layer, it encodes partially known causal parent node relationships into network connection rules as hard constraints. This ensures causal fidelity while unleashing the function approximation capability of deep networks. Validation on the Sachs real-world dataset demonstrates that embedding causal knowledge effectively prevents overfitting, and the learned causal functions are interpretable.

Finally, conclusions and future prospects are presented. The DAG-KAN and Causal-KAN methods proposed in this thesis systematically advance the research paradigm of causal discovery from "causal existence judgment" to "causal mechanism analysis". They construct a new paradigm for causal function learning that combines mathematical transparency with practical robustness, providing actionable mechanism-level causal knowledge for fields requiring an understanding of "how to intervene", such as precision medicine and policy intervention.

\end{eabstract}
